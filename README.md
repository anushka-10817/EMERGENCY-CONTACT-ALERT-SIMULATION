# EMERGENCY-CONTACT-ALERT-SIMULATION

# OBJECTIVE
The Emergency Contact Alert Simulation system is a simple Python - based program designed to send alerts to the family members in case of emergencies. It is essential in schools, colleges, parks, workplaces, etc. The system allows users to add family members contact numbers, choose different types of alerts, and simulate the sending of notifications during any emergency. If you are stuck in an unknown place or you are travelling somewhere alone and you are feeling unsafe you can immediately send an emergency alert to your family members. This system helps demonstrate how real- world safety alert systems work in a basic, easy to use and understand manner. 

# FEATURES
Content Information: Easy to view, add, and remove emergency contact numbers.
Trigger Alert: Send a programmed alert message to all saved contact numbers.
Pre-defined Messages: Choose from given common emergency messages.
Custom Message: Quote a custom alert message.
Activity Log: View a history of the last three triggered alerts.
Date Stamping: Records the date stamp of the alert.
